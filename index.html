PC·휴대폰에 같은 repo·토큰을 넣으면 자동으로 일치합니다. 기존 한투 데이터는 예전에 정한 동기화 암호로 암호화돼 있으니, 그 암호를 한 번 입력하면 복구됩니다. 처음 시작하거나 새 데이터면 비워두세요.

고객 정보

영업·CRM 정보를 직접 입력합니다. 자산 엑셀과 별도로 이 기기에 저장되고, 동기화·백업에 함께 실립니다.

📋 텍스트 붙여넣기로 자동 채우기

고객 정보 텍스트(연락처·주소·주민번호 블록 또는 「고객명 / 성별 / 연락처 …」 라벨 형식)를 붙여넣고 자동 채우기를 누르면 아래 항목이 채워집니다. 이메일·상령일·Insu/User/GA No.·배우자 등은 비고에 정리됩니다.

✓ 계좌 확인 · 색 채움=잔고 있음 · 테두리만=0원 계좌 · 색 없음=실계좌 없음

🛡 보험 계약 추가

가입보험 계약을 직접 입력하거나, 토스 ERP 표·정리메모를 붙여넣거나, 엑셀 파일로 가져옵니다. 계약자별로 자동 분류·중복 통합됩니다.

토스 ERP '체결 보험' 표 · 또는 「2023.09.08 안소연(안소연) DGB생명 상품명 (월납 1,000,000원)」 형식. 여러 줄·여러 명 한번에 가능.

엑셀/CSV(.xlsx .xls .csv)는 자동 인식합니다. PDF·이미지는 아직 자동 인식이 안 되니 내용을 붙여넣기 탭에 넣어주세요.

파일을 드래그하거나 클릭.xlsx · .xls · .csv
